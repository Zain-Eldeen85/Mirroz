# **App Name**: Mirroz Booking

## Core Features:

- Mirror Showcase: Display a grid of available mirrors with preview images and brief descriptions.
- Mirror Detail View: Showcase mirror specifications (size, design, availability) and high-resolution previews with zoom capability.
- Booking Form: Implement a booking form with date picker, time slot selector, and quantity selector.  It will be designed for ease of use and quick booking.
- Internationalization: Support for English and Arabic, with Arabic displaying RTL.
- Theme Toggle: Enable switching between light and dark themes with preference persistence.
- Skeleton Loaders: Implement skeleton loaders for main views to improve UX during loading times. It will show mirror-style placeholders with smooth transitions to content.

## Style Guidelines:

- Background: Light mode (#f8f9fb) for a clean, open feel, dark mode (#0e1116) for a premium, subdued ambience.
- Surface: In light mode, a semi-transparent white (rgba(255,255,255,0.6)); in dark mode, a subtle transparent white (rgba(255,255,255,0.05)).
- Primary: Light gray-blue (#cfd8e3) in light mode, a similar but lighter gray-blue (#9fb3c8) in dark mode for balance and elegance. Converted to RGB: Light Mode (#CFCFD4), Dark Mode (#9FACC2).
- Accent: Light blue-gray (#e6edf7) in light mode, a darker shade of blue-gray (#1b2430) in dark mode for subtle highlighting.
- Font pairing: 'Poppins' (sans-serif) for headlines, providing a geometric, fashionable touch, and 'PT Sans' (sans-serif) for body text for a humanist, modern, warm touch.
- Sizes: Use sizes like h1 (text-4xl font-semibold), h2 (text-2xl font-medium), body (text-base), small (text-sm).
- Minimalist line icons to represent mirror features and booking options.
- Mirror-inspired layout with glassmorphism, blur, transparency, and rounded shapes.
- Subtle hover reflections and soft shine animations for a luxury feel.
